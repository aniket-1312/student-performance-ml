# Student Performance Classification (Machine Learning)

This project predicts student grade class using machine learning.

Dataset Features:
- StudyTimeWeekly
- Absences
- GPA
- ParentalSupport
- Activities

Models Used:
- Logistic Regression
- Decision Tree
- Random Forest

Accuracy: ~90%

Tools:
Python, Pandas, Scikit-Learn, Matplotlib, Seaborn
